lrhiero
========

This is a left-to-right decoder for hierarchical phrase-based (hiero) SMT system.


# Usage
```
python decoder_cp.py --config <lrhiero.ini> --inputfile <test.in> --outputfile <test.out> --ttable-file <test.rule>
```

# Requirements
* Python 2.6 or higher
